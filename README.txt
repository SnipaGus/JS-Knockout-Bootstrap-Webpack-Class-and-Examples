1) Download and install NodeJS from https://nodejs.org/en/. Ensure that you use version 8.12 or higher.
2) Open the 'Node.js command prompt' and navigate to this project folder.
3) Execute the command 'npm install'. This will install the libraries that this project depends on.
4) Execute the command 'npm start'. This will compile/bundle your source code and make it availible at web address 'localhost:8080'.
5) Example pages are availible at 'localhost:8080/example1.html' and 'localhost:8080/example2.html'
